# Execution-Context
실행 컨텍스트 동작 방식을 그린 자료입니다.   
실행 컨텍스트를 입체적으로 이해하고, 실제 코드를 작성하면서도 머릿속으로 그려낼 수 있도록 훈련하였습니다.   
이를 통해 코드를 읽으면서 혼선을 줄이고, 명확하게 동작 원리를 설명할 수 있게 되었습니다.
